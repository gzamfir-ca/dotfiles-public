# Dotfiles public repository

### Purpose

- Hosts various development environment configuration files

### Usage

- Requires homebrew, Z shell, git, iterm2 & VS Code installed

```
❯ cp Brewfile ~
❯ cp .zshrc ~
❯ cp .gitconfig ~
❯ cd ~
❯ brew bundle install
```

- iTerm2 open General -> Preferences -> load the settings from com.googlecode.iterm2.plist
- VS Code open Settings -> Open Settings (JSON) -> then copy and paste from settings.json
- VS Code open Command Pallete, type shell command -> use install code command in path
- VS Code open the extensions file -> then copy, paste and run any of code install commands
